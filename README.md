# Titanic Data Analysis with Pandas

## Overview

This project presents an exploratory data analysis (EDA) of the Titanic dataset using Python and the Pandas library. The goal is to clean, analyze, and extract meaningful insights from real-world data.

---

## Objectives

Create and structure a custom dataset using Python dictionaries
Perform data cleaning and preprocessing
Conduct exploratory data analysis (EDA)
Analyze survival patterns based on gender, class, and age
Derive meaningful insights from the dataset

---

## Tools & Technologies

Python
Pandas
Google Colab

---

## Key Steps Performed

Loaded and explored the dataset using `head()`, `info()`, and `describe()`
Handled missing values and removed irrelevant data
Applied grouping and aggregation techniques
Filtered specific subsets for deeper analysis
Categorized age into groups for better understanding

---

## Key Insights

Females had a significantly higher survival rate than males
First-class passengers had better survival chances
Children were more likely to survive compared to adults
Survival was strongly influenced by gender, class, and age

---

## Project Structure

`Task_3_Titanic_Data_Analysis.ipynb` → Main analysis notebook
`train.csv` → Dataset used for analysis

---

## Conclusion

This project demonstrates how data preprocessing and exploratory analysis can be used to uncover patterns and insights from real-world datasets.
